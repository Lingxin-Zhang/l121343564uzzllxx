# review_gpt

本目录用于和负责代码仓库 review 的 GPT 沟通本轮初始化工作。

建议 review GPT 先阅读：

1. `latest.md`：最新一轮修改摘要、测试结果、已知问题和下一步建议。
2. 对应的 `round_xx_summary.md`：该轮完整交接说明。
3. `review_request.md`：项目初始化需求、实际完成内容、审查重点。
4. `../AGENTS.md`：项目目标、论文边界、后续 agent 必须遵守的规范。
5. `../README.md`：仓库简介、结构说明和当前状态。
6. `../paper/notes/`：论文故事、实验计划、创新边界和方法占位。

本轮仓库仍处于 skeleton 阶段。请不要要求或假设已经存在完整 backend、benchmark、实验结果、speedup 数字或 BER 结论。
